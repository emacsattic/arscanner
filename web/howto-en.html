Author rights Scanner Howto

Installation :

  1. Download and save the tarball somewhere on your computer.
  2. Extract the content of the tarball:
    “tar xvfz arscanner-0.1.tgz”
  3. type “make”
  4. type “sudo make install”
  5. add in your .emacs file the following line :
    (autoload 'arscanner "arscanner"
    "Manage copyright holders and license in source file header" t)

Arscanner consists in two files. “arscanner.el” is a emacs lisp file for the arscanner mode. The other is a binary file arscanner, compiled from arscanner.l a flex file, for extracting copyright lines and recognizing the licenses.

I don't presently use an Emacs bovinator like Semantic but it would be useful to eliminate dependencies on C code and Flex.

How to use arscanner :

To set some parameters 

M-x ars-set-exclude-extension : Sets the extensions of files to ignore
M-x ars-set-include-extension : Sets extensions of sources file

To open an arscanner buffer

M-x arscanner and type the path to the root of your source tree. A buffer containing a list of files from the source tree is opened in Arscanner mode.

On a file name's line

[return],v : Opens the source file in view mode
e : Opens the source file in edit mode

To add a header :

t : Puts the content of the yank buffer as header for the file  on the current line. If the current line is a directory, then the process is done recursively on all the files with a source file's extension (see ars-set-include-extension), contained in the directory.
