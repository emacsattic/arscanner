Author rights Scanner Howto

Installation :

  1. Télécharger et sauver quelque part l'archive
  2. Extraire le contenu de l'archive :
    " tar xvfz arscanner-0.1.tgz "
  3. taper " make "
  4. taper " sudo make install "
  5. Ajouter dans votre fichier .emacs la ligne suivante :
    (autoload 'arscanner "arscanner" "Manage copyright holders and license in source file header" t)

Arscanner est composé de deux fichiers. " arscanner.el " est un fichier Emacs Lisp pour le mode Emacs arscanner. L'autre est " arscanner.l " un lexer pour extraire les lignes de copyright et reconnaître les licences.

Actuellement je n'utilise pas " bovinator " pour Emacs comme Semantic mais ce serait intéressant pour éliminer la dépendance avec le code C et Flex.

Comment utiliser Ascanner :

Pour fixer quelques paramètres :

M-x ars-set-exclude-extension : Définit les extensions des fichiers à ignorer

M-x ars-set-include-extension : Définit les extensions des fichiers source

Pour ouvrir un buffer arscanner :

M-x arscanner et taper le chemin de votre arborescence de codes source. Un buffer contenant la liste des fichiers de l'arborescence source, s'ouvre en mode Arscanner.

Sur la ligne d'un nom de fichier :

[return],v : ouvre en mode vue le fichier

e : ouvre en mode édition le fichier

Pour ajouter un entête :

t : Pose le contenu du presse papier (yank buffer) comme entête pour le fichier sur la ligne courante. Si la ligne courante est un répertoire, alors le processus est réalisé récursivement sur tous les fichiers avec une extension de fichiers source (voir ars-set-include-extension), contenu dans le répertoire.
